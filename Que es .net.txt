What is .NET?
.NET is a complete ecosystem for developing and deploying applications for desktops, mobile devices, the cloud, 
and IoT devices. Developers use programming languages likes Visual Basic, C#, and F# to build with .NET.
You can use .NET tools on the command line, or embedded in applications like Visual Studio or Visual Studio Code,
to compile your code into a running application.

What is Razor?
Razor is a format for generating text-based content, like HTML. Razor files have a cshtml or a razor file extension, 
and contain a mix of C# code along with HTML.

What is Blazor?
Blazor is a user-interface framework built on .NET and Razor. Blazor applications can run on a server as part of an 
ASP.NET application, or can be deployed to run in the browser on a user's machine similar to a single-page-application.
